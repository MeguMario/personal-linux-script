
# personal-linux-script

A collection of script that I'm using on my Linux installation

## darkswitch

Handling Dark/Light mode switching on GNOME for qt/KDE apps, apps that need special handling (Waydroid, Duckstation AppImage, Krita and Kdenlive) and some extensions, use in conjunction with [Night Theme Switcher extension](https://extensions.gnome.org/extension/2236/night-theme-switcher/). Relies heavily on [Libadwaita-KDE theme](https://github.com/MeguMario/Libadwaita-KDE)

## arch-alt-update
This automate system update on Arch Linux alongside updating the [patched version of GNOME](https://github.com/aunetx/blur-my-shell/issues/594) that I have in addition to rebuilding [qadwaitadecorations](https://github.com/FedoraQt/QAdwaitaDecorations) with every system update and cleaning up pacman once update is finished
